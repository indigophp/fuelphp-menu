**This package is a wrapper around [KnpLabs/KnpMenu](https://github.com/KnpLabs/KnpMenu) package.**


## Install

Via Composer

``` json
{
    "require": {
        "indigophp/fuelphp-menu": "@stable"
    }
}
```
